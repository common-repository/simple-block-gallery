Add the effect of Masonry and Slider to images.

== Description ==

= Only HTML & CSS =
* Create an HTML and CSS only block. The display does not collapse even if the plugin is deactivate.

= Masonry settings =
* Width
* Space
* Rounded Images

= Slider settings =
* Interval
* Note: Since the CSS class and @keyframes specify the ID of the image called first in the gallery, if the same image is specified as the first image of multiple sliders in the same post, the display will be distorted.

= How it works =
[youtube https://youtu.be/1OIIhZfmvVA]

== Installation ==

1. Upload `simple-block-gallery` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
